# CurrencyConverter
A Java-based Currency Converter application that allows users to convert an amount from one currency to another using predefined exchange rates. The program supports multiple currencies such as USD, INR, EUR, GBP, JPY, AUD, CAD, and CNY, and includes input validation for currency codes and amount values. 

## Features

- Convert between multiple currencies
- Supports USD, INR, EUR, GBP, JPY, AUD, CAD, and CNY
- User-friendly console interface
- Input validation for currency codes and amount values
- Displays accurate converted amounts clearly

## Technologies Used

- Java
- HashMap
- Scanner Class
- Arithmetic Operations

## Concepts Used
Java Collections (HashMap): Used to store and manage predefined currency exchange rates.
Scanner Class: Used to take user input from the console.
Conditional Statements: Used to validate currency codes and handle user choices.
Loops: Used to repeatedly prompt the user until valid input is provided.
Methods: Used to organize the program into reusable and structured functions.
Arithmetic Operations: Used to calculate the converted currency amount.
Input Validation: Ensures that the entered currency code and amount are valid.
